# Bridge Up Marketing Site

Marketing website for Bridge Up - a real-time bridge status app for the St. Lawrence Seaway region.

## About

Bridge Up provides real-time bridge status and predictive reopening times for bridges in St. Catharines, Port Colborne, Montreal, and Salaberry-de-Valleyfield. This repo contains the marketing website.

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Framer Motion

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
```
